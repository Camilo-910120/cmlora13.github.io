En HTML5 no es necesario proporsionar valores para todos lo atributos, como los controles, reproduccion automatica y atributos de bucle utilizados con el elemento <video> estos atributos son como interruptores de encendido/apagado. si el atributo esta presente, activa esta opcion. si se omite el atributo, la opcion esta desactivada. Si el navegador no admite el elemento <video> o el formato de video utilizado, se mostrara lo que este entre la <video> de apertura y las etiquetas de </video> de cierre.

none:el navegagor no debe cargar el video hasta que el usuario presiona play.
auto:el navegador debe descargar el video cuando se carga la pagina.
