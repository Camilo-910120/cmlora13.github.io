Cada elemento HTML puede llevar el atributo id. se utiliza para identificar de forma unica ese elemento de otros elementos de la pagina. Su valor debe comenzar con una letra o un gion bajo (no un numero o cualquier otro caracter). es importante que no hay dos elemntos en la misma pagina que tengan el mismo valor para sus atributos ID (de lo contrario,el valor noes unico)

Dar a un elemento una identidad unica le permite darle un estilo diferente que cualquier otra instancia del mismo elemento en la pagina.

Por ejemplo, es posible que desees asignar un parrafo dentro de la pagina (quizas un parrafo que contiene una cotizacion de extraccion) un estilo diferente que todos los demas parrafos. En el ejemplo, el parrafo con el atributo ID cuyo valor es pullquote se hace en mayusculas usando css.

Class Atribute

Cada elemento de HTML tambien puede llevar un atribudo de clase. Aveces, en lugar de identificar de forma unica dentro de un documento, deseara de una manera de identificar varios elementos como diferentes de los demas elementos de la pagina. por ejemplo, es posible que tengan algunos parrafos de texto que contengan informacion que sea mas importante que otros y desee distinguir estos elementos o puede que desee diferenciar entre vinculos que apuntan a otras paginas de otro sitio y vinculos que apuntan a sitios externos.

Para ello, puede utilizar el atributo de clase. su valor debe describir la clase a la que partenece. En el ejemplo de la izquierda, los parrafos clave tienen un atributo de clase cuyo valor es importante. El atributo de clase en cualquier elemento puede compartir el mismo valor. por lo tanto, en este ejemplo, el valor de Important tambien se podria utilizar en encabezados y viculos.

De forma predeterminada, el uso de estos atributos no afecta a la presentacion de un elemento. Solo cambiara su apariencia si hay una regla de CSS que identificar que debe mostrarse de forma diferente. En este ejemplo, el CSS se ha aplicado para hacer los elementos con un atributo de clase cuyo valor es mayuscula importante, y los elementos con un atributo de clase cuyo valor es la admision roja. Si desea indicar que un elemnto pertenece a varias clases, puede separar los nombres de clase con un espacio.

Grouping block elements

El elemento <div> permite agrupar un grupo de elementos juntos en un cuadro de nivel de bloque. por ejemplo, puede crear un elemento <div> para que contenga todos los elementos del encabezado del sitio (el logotipo y la navegacion), O puede crear un elemento <div> para que contenga informacion en bloques.

En un navegador el contenido del elemento <div> comenzara en una nueva linea, pero aparte de esto no hara ninguna diferencia en la presentacion de la pagina. Sin embargo el uso de un atributo ID o class en el elemento <div> significa que puede crear reglas de estilo CSS para indicar cuanto espacio debe ocupar el elemento <div> en la pantalla y cambiar la apariencia de todos los elementos contenidos en el.Tambien puede facilitar el seguimiento del codigo si ha utlizado elementos <div> para contener cada seccion de la pagina.

Dado que puede haber varios otros elementos dentro de un elemento <div> puede ser util agregar un comentario despues de la etiqueta </div> de cierre esto le permite ver claramente a que etiqueta de apertura se supone que corresponde.
